# Percentage Calculator

Simple web-based percentage calculator with four modes:
- "What is % of ?" (X% of Y)
- "What percent is A of B?"
- "Increase / Decrease by %"
- "Percentage change from A to B"

Usage:
1. Open `index.html` in a browser.
2. Choose a mode, enter values, and click Calculate (or press Enter).
3. Use Copy to copy the result to clipboard.

This repository contains separate HTML, CSS, and JS files.
